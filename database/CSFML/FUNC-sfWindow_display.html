sfWindow_display

(C/CSFML/sfWindow_display)

Prototype

void sfWindow_display(sfRenderWindow * renderWindow)

Description


Displays a render window on screen.

This function will update and display everything that has been added to the window buffer.

Parameters

renderWindow
The window you want to update

Example

In this example, we create a window, then update it using the function.

      #include <stdlib.h>
     #include <CSFML/Window/Window.h>
    
     int main ( )
     {
         sfWindow *window = sfWindow_create ( { 1600 , 900 , 32 } , "My Window" , 0 , NULL ) ;

         sfWindow_display ( window ) ;

         return ( 0 ) ;
     }
