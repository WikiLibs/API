sfWindow_isOpen

(C/CSFML/sfWindow_isOpen)

Prototype

sfBool sfWindow_isOpen(sfRenderWindow * renderWindow)

Description


Returns a sfBool indicating if the window is open.

This function will return a sfTrue if the window is open, sfFalse otherwise.

Parameters

renderWindow
The window you want to check

Example

In this example, we create a window, then we use twice the function, to demonstrate how it works.

      #include <stdio.h>
     #include <stdlib.h>
     #include <CSFML/Window/Window.h>
    
     int main ( )
     {
         sfWindow *window = sfWindow_create ( { 1600 , 900 , 32 } , "My Window" , 0 , NULL ) ;

         if ( sfWindow_isOpen ( window ) == sfTrue )
                 printf ( "The window is open\n" ) ;
         else
                 printf ( "The window isn't open" ) ;

         sfWindow_destroy ( window ) ;

         if ( sfWindow_isOpen ( window ) == sfTrue )
                 printf ( "The window is open\n" ) ;
         else
                 printf ( "The window isn't open" ) ;

         return ( 0 ) ;
     }
