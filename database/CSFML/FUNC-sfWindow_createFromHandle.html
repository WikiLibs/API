sfWindow_createFromHandle

(C/CSFML/sfWindow_createFromHandle)

Prototype

sfWindow * sfWindow_createFromHandle(sfWindowHandle handle, const sfContextSettings* settings)

Description


Construct a window from an existing control















Use this constructor if you want to create an OpenGL
rendering area into an already existing control.

The second parameter is a pointer to a structure specifying
advanced OpenGL context settings such as antialiasing,
depth-buffer bits, etc.

Parameters

handle
Platform-specific handle of the control
settings
Additional settings for the underlying OpenGL context
