sfWindow_create

(C/CSFML/sfWindow_create)

Prototype

sfWindow * sfWindow_create(sfVideoMode mode, const char * title, sfUint32 style, const sfContextSettings * settings)

Description


Construct a new window.

This function creates the window with the size and pixel depth defined in mode. An optional style can be passed to customize the look and behaviour of the window (borders, title bar, resizable, closable, ...). If style contains sfFullscreen, then mode must be a valid video mode.

The fourth parameter is a pointer to a structure specifying advanced OpenGL context settings such as antialiasing, depth-buffer bits, etc.

Parameters

mode
Video mode to use (defines the width, height and depth of the rendering area of the window)
title
Title of the window
style
Window style
settings
Creation settings (pass NULL to use default values)

Example

In this example, we just create a window using the function. This code only displays the window for one frame

      #include <stdlib.h>
     #include <CSFML/Window/Window.h>
    
     int main ( )
     {
         sfVideoMode mode = { 1600 , 900 , 32 } ;
         const char *title = "My Window"
    
         sfWindow *window = sfWindow_create ( mode , title , 0 , NULL ) ;

         return ( 0 ) ;
     }
