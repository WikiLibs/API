sfWindow_destroy

(C/CSFML/sfWindow_destroy)

Prototype

void sfWindow_destroy(sfWindow* window)

Description


Destroy a window.

This function will destroy a previously created window (using sfWindow_create()), and free the resources taken.

Parameters

window
Window to destroy

Example

In this example, we just create a window using sfWindow_create, then we destroy it. This code only displays the window for one frame

      #include <stdlib.h>
     #include <CSFML/Window/Window.h>
    
     int main ( )
     {
         sfWindow *window = sfWindow_create ( { 1600 , 900 , 32 } , "My Window" , 0 , NULL ) ;

         sfWindow_destroy ( window ) ;

         return ( 0 ) ;
     }
